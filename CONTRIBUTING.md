# Contributing to Awesome Electrobun

Thanks for taking the time to contribute!

## What to add

We accept:
- Electrobun templates/starter kits
- Plugins/tools that improve the Electrobun dev experience
- Example apps (open source)
- High-quality blog posts, talks, or guides (prefer primary sources)
- CI/CD workflows, signing/notarization guides

## What not to add

- Low-quality SEO content or link farms
- Duplicates of primary sources
- Broken links

## Pull requests

- Keep PRs small and focused.
- Place the entry in the most relevant category.
- If you're adding a new category, explain why.
